# Portfolio

This is a Vite + React personal portfolio template prepared for static hosting on Vercel.

## Edit your content

Update all personal details from:

- `src/data/portfolio.js`

That file controls:

- name, headline, email, and social links
- skills
- project cards
- about text
- contact content

Your resume file is served from:

- `public/resumee.pdf`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

This project is a normal static Vite app. No backend or environment variables are required for the current version.
